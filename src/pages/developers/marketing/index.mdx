# Marketing

## Badge

You can use the following badge to advertise your software's presence on Overt.

<img src="/images/overt-badge.png" width="256" class="d-block mx-auto my-4" />

For automatic artwork updates, link to this image from your site and/or README. For example:

```html
<img
  src="https://getovert.app/images/overt-badge.png"
  width="240"
  alt="Get it on Overt, an open app store"
/>
```

## Quick-install links

Users can install your software via Overt by clicking a link on your website or project page.

Use the following tool to generate a quick-install link for your software.

```html
<a href="YOUR LINK HERE">
  <img
    src="https://getovert.app/images/overt-badge.png"
    width="240"
    alt="Get it on Overt, an open app store"
  />
</a>
```
