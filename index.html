OpenStore Beta

An open app store
for macOS


OpenStore helps you find, install, and manage your favorite apps.

OpenStore doesn't:

  • Restrict app features
  • Process payments

OpenStore does:

  • Give you a catalog of 3000+ apps at your fingertips
  • Install, update, and uninstall apps with a single click
  • Let you add your own source repositories
Install Source Code on GitHub

Installation currently requires that you manually install Homebrew and add OpenStore's "tap."

OpenStore is powered by Homebrew, a package manager for macOS. The default repository is graciously maintained by the Homebrew user community.

While OpenStore currently only integrates with Homebrew, it will be extended in the near future to support other package managers, especially on Windows.
